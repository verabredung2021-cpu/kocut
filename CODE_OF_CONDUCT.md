# Code of Conduct

KoCut aims to be a practical and respectful open-source project.

Expected behavior:

- Be respectful and constructive
- Keep discussions focused on the project
- Report bugs with useful reproduction details
- Do not post private media, patient/client data, API keys, or copyrighted production material
- Give feedback in a way that helps improve the project

Unacceptable behavior:

- Harassment or personal attacks
- Posting private or sensitive data
- Spam or unrelated promotion
- Abusive language toward contributors or users

Maintainers may remove comments, issues, or pull requests that violate these expectations.
