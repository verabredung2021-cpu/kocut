# Support

KoCut is an early-stage open-source project.

## Questions

For general questions, ideas, or workflow discussion, use GitHub Discussions if enabled.

## Bugs

For reproducible bugs, open a GitHub issue using the bug report template.

Please include:

- KoCut version
- Operating system
- Python version
- FFmpeg version
- Command used
- Error message
- Minimal reproduction steps

Do not upload private video, audio, patient/client media, API keys, or copyrighted production material.

## Feature requests

For new workflow ideas, use the feature request template.

Good feature requests describe:

- The editing problem
- The desired behavior
- The target editor or workflow
- Example input/output when safe to share
