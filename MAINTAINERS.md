## Maintainer focus

Current maintainer focus:

- Korean subtitle and cut suggestion workflow
- Local-first video processing
- SRT, EDL, FCPXML, JSON, and Markdown exports
- Windows creator workflow
- Security review for file paths, exports, dependencies, and subprocess calls
- Documentation and example outputs

## Release process

1. Update tests
2. Update README if behavior changes
3. Update CHANGELOG
4. Run `python -m pytest`
5. Commit with a release message
6. Push to `main`
7. Create a version tag
