# Release Checklist

Use this checklist before publishing a new KoCut release.

- [ ] Run `python -m pytest`
- [ ] Check CLI command behavior
- [ ] Check GUI launch behavior if changed
- [ ] Check SRT export
- [ ] Check EDL export
- [ ] Check FCPXML export if changed
- [ ] Check JSON/Markdown export
- [ ] Review file path handling
- [ ] Update README
- [ ] Update CHANGELOG
- [ ] Create a Git tag
- [ ] Push tag to GitHub
